LIB2024- QA

  1. Question: What are the three components of Web?

    Answer: The three components of Web are HTML, CSS, and JavaScript.

  2. Question: Define version control and give one example.

    Answer: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. GitHub is one example of version control.

  3. Question: What tag is used to create the title of a web page? Give an example.

    Answer: <title> is used to create the title of a web page. For example: <title>LIB2024 - QA</title>.

  4. Question: What tag is used to create a paragraph in html? Give an example.

    Answer: <p> is used to create a paragraph in html. For example: <p>This is a paragraph</p>.

  5. Question: Identify the odd one from these tags: <span>, <p>, &hta>, <i> and state the reason?

    Answer: The odd one from these tags is <p> because <p> is a block level element while the others are inline elements.

  6. Question: What are the tags required to create a navigation list in HTML?

    Answer: <nav> as a container of a navigation list, <ul> or <ol> is used to create an unordered or ordered navigation list, <li> is used to create a list item, and <a> is used to create links in a HTML document.

  7. Question: What element is used to create images? Name its attributes.

    Answer: <img> is used to create images. Its attributes are src, alt, width, and height.

  8. Question: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Answer: The odd one from these tags is <b> because <b> is used to make text bold while the other tags are used in creating a table.

  9. Question: Give an example of color picker input.

    Answer: <input type="color" id="favcolor" name="favcolor" value="#ff0000">

  10. Question: What is the use of name and value attributes? Give two examples.

    Answer: The name attribute specifies a name for the element. For form elements, the name attribute also used as a reference when the data is submitted. The value attribute specifies the value of an <input> element.
    Example 1: <input type="radio" id="female" name="gender" value="female">
    Example 2: <input type="text" id="fname" name="fname" value="Chloe">

  11. Question: Give an example of inline style.

    Answer: <h1 style="color:blue;">This is example of inline style</h1>

  12. Question: How would you link an external style sheet? Give example.

    Answer: I would include a reference to the external style sheet file inside the <link> element, inside the head section in the HTML page. For example, <link rel="stylesheet" type="text/css" href="style.css">
