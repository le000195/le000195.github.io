LIB2024- QA

  1. Question: What are the three components of Web?

    Answer: The three components of Web are HTML, CSS, and JavaScript.

  2. Question: Define version control and give one example.

    Answer: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. GitHub is one example of version control.

  3. Question: What tag is used to create a paragraph in html? Give an example.

    Answer: <p> is used to create a paragraph in html. For example: <p> This is a paragraph </p>.

  4. Question: Identify the odd one from these tags: <span>, <p>, &hta>, <i> and state the reason?

    Answer: The odd one from these tags is <p> because <p> is a block level element while the others are inline elements.

  5. Question: What are the tags required to create a navigation list in HTML?

    Answer: <nav> as a container of a navigation list, <ul> or <ol> is used to create an unordered or ordered navigation list, <li> is used to create a list item, and <a> is used to create links in a HTML document.

  6. Question: What element is used to create images? Name its attributes.

    Answer: <img> is used to create images. Its attributes are src, alt, width, and height.

  7. Question: Identify the odd one from these tags: <table>, <tr>, <b>, <td&Gt and state the reason.

    Answer: The odd one from these tags is <b> because <b> is used to make text bold while the other tags are used in creating a table.
